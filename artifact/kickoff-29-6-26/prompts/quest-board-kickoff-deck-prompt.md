# Quest Board kickoff deck prompt

Create a **clean, modern, 5-slide kickoff deck** introducing Quest Board as Build's first quest.

Audience constraints:
- Mixed skill levels, including first-time open-source contributors.
- Explain concepts in plain language.

Deck requirements:
1. Cover slide: Quest Board one-liner and relationship to Build.
2. Why it exists: "what should we build next and who can help?"
3. How it works: text-first source of truth, generated artifacts.
4. Contribution paths by skill level: beginner to advanced.
5. First two-week kickoff plan and clear call to action.

Design requirements:
- 16:9 layout.
- Clean visual system aligned with Build deck style.
- Consistent structure and strong readability.

Source context to reflect:
- Quest Board README from github.com/The-Last-Founder/quest-board.
- GitHub-native workflows: PRs, issues, validation, generation.

Output:
- A print-ready PDF.
