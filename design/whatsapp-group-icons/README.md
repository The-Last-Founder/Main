# WhatsApp group icon concepts

The primary Build WhatsApp group icon lives in this folder, with the remaining concepts kept as alternatives.

## Primary asset

- `build-whatsapp-icon.svg` — selected main icon (formerly concept #4, "launch path")
- Format: SVG
- Dimensions: 1024×1024
- Crop target: designed to survive WhatsApp's circular crop with a centered safe area
- Style: minimal, calm, collaborative, slightly game/quest-inspired

## Preview

![Main WhatsApp group icon](./build-whatsapp-icon.svg)

## Alternatives

Other concepts are preserved in [`alternatives/`](./alternatives):

- `build-whatsapp-icon-01-quest-board.svg` — quest-board grid with one highlighted tile
- `build-whatsapp-icon-02-party-formation.svg` — three collaborators orbiting an open slot
- `build-whatsapp-icon-03-blueprint.svg` — project brief / blueprint with a completion mark
- `build-whatsapp-icon-05-builder-spark.svg` — two build blocks under a shared spark
